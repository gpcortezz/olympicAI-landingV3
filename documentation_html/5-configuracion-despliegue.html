5.1. Requisitos del Sistema:

  • Node.js (versión compatible con el proyecto)
  • npm/yarn/pnpm

5.2. Variables de Entorno:

No especificado

5.3. Pasos de Instalación (Entorno de Desarrollo):

  1. Clonar el repositorio
  2. Ejecutar npm install
  3. Ejecutar npm run dev para iniciar servidor de desarrollo

5.4. Pasos de Despliegue (Entorno de Producción):

  1. Ejecutar npm run build
  2. Desplegar contenido de la carpeta dist

5.5. Scripts de Ejecución Comunes:

  • dev: Inicia servidor de desarrollo
  • build: Compila para producción
  • preview: Previsualiza compilación de producción
