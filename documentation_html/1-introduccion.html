1.1. Nombre del Proyecto

OlympicAI Landing V3

1.2. Descripción General

OlympicAI Landing V3 es una landing page moderna y atractiva desarrollada específicamente para promocionar las Olimpiadas Universitarias de Inteligencia Artificial 2025. La aplicación presenta un diseño futurista con efectos visuales avanzados, un contador regresivo interactivo y una experiencia de usuario optimizada para dispositivos móviles y de escritorio.

El proyecto se enfoca en crear una experiencia visual impactante que capture la atención de los estudiantes universitarios interesados en competir en desafíos de inteligencia artificial, machine learning y algoritmos innovadores.

1.3. Visión General del Negocio/Dominio

El proyecto pertenece al sector educativo y tecnológico, específicamente en el ámbito de las competencias universitarias de inteligencia artificial. Las olimpiadas universitarias de IA representan una oportunidad para que estudiantes de diferentes universidades demuestren sus habilidades en:

  • Machine Learning: Desarrollo de algoritmos de aprendizaje automático
  • Procesamiento de Datos: Análisis y manipulación de datasets complejos
  • Algoritmos: Resolución de problemas computacionales
  • Innovación Tecnológica: Creación de soluciones innovadoras

La landing page sirve como punto de entrada principal para informar sobre el evento, generar expectativa y facilitar el registro de participantes.

1.4. Audiencia

Esta documentación está dirigida a:

  • Desarrolladores Frontend: Para entender la arquitectura Vue.js, configuración de Vite y implementación de efectos visuales
  • Diseñadores UX/UI: Para comprender las decisiones de diseño, responsividad y experiencia de usuario
  • Administradores de Sistemas: Para el despliegue, configuración y mantenimiento de la aplicación
  • Gerentes de Proyecto: Para entender el alcance, funcionalidades y tecnologías utilizadas
  • Desarrolladores Full-Stack: Para integrar esta landing page con sistemas backend futuros

1.5. Características Principales

La aplicación incluye las siguientes características destacadas:

  • Diseño Futurista: Interfaz moderna con efectos de partículas y animaciones avanzadas
  • Contador Regresivo: Timer interactivo que muestra el tiempo restante hasta el evento (18 de septiembre de 2025)
  • Responsividad Completa: Optimizada para dispositivos móviles, tablets y desktop
  • Efectos Visuales: Sistema de partículas con tsParticles, animaciones GSAP y efectos de hover
  • Performance Optimizada: Carga rápida y experiencia fluida
  • Accesibilidad: Diseño que considera diferentes tipos de usuarios

1.6. Objetivos del Proyecto

Los objetivos principales de esta landing page son:

  • Generar interés y expectativa sobre las Olimpiadas Universitarias de IA 2025
  • Proporcionar información clara sobre el evento y sus características
  • Crear una experiencia visual memorable que refleje la innovación tecnológica
  • Facilitar el proceso de registro y participación de estudiantes
  • Establecer una presencia digital profesional para el evento
