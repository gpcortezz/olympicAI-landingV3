Información del Proyecto

Este proyecto es una landing page moderna y atractiva desarrollada para promocionar las Olimpiadas Universitarias de Inteligencia Artificial 2025. La aplicación presenta un diseño futurista con efectos visuales avanzados, un contador regresivo interactivo y una experiencia de usuario optimizada.

Vue.js 3 Vite Tailwind CSS GSAP tsParticles JavaScript ES6+
